# The Bricks

`The Bricks` je preprosta arkadna igra v brskalniku, narejena z uporabo `HTML`, `CSS`, `JavaScript`, `jQuery` in `HTML5 Canvas`.

Igralec upravlja ploščico, odbija žogico, uničuje bloke in napreduje skozi več stopenj z različno težavnostjo.

## Opis

Igro zaženeš neposredno v brskalniku tako, da odpreš datoteko `index.html`.

## Funkcionalnosti

- 3 stopnje z naraščajočo težavnostjo
- prikaz točk
- prikaz časa
- prikaz trenutnega levela
- naključne barve in vzdržljivost blokov
- premikanje ploščice s tipkovnico
- obvestilo o zmagi s knjižnico `SweetAlert2`

## Uporabljene tehnologije

- HTML5
- CSS3
- JavaScript
- jQuery
- HTML5 Canvas
- SweetAlert2

## Zagon projekta

1. Prenesi ali kloniraj repozitorij.
2. Odpri mapo projekta.
3. Zaženi `index.html` v brskalniku.

Za delovanje niso potrebna dodatna orodja ali namestitev paketov.

## Kontrole

- `Leva puščica`: premik ploščice levo
- `Desna puščica`: premik ploščice desno

## Struktura projekta

```text
The-Bricks-main/
├── index.html
├── css/
│   └── style.css
└── js/
    ├── jQuery.js
    └── script.js
```

## Potek igre

Cilj igre je uničiti vse bloke na zaslonu, ne da bi žogica padla mimo ploščice.
Ko igralec počisti vse bloke, napreduje na naslednji level, kjer je igra zahtevnejša.
